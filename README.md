# RecyclerViewWithDynamicData
This application shows how to fetch the huge data from server, parse the data and fill into RecyclerView with lazy loading.
Components used :
1. Kotlin
2. Retrofit
3. MVVM
4. RxKotlin
5. Dagger
6. RecyclerView
7. Navigation
8. LiveData
9. Pull to refresh
10. Log4k
11. Picasso to load the image
